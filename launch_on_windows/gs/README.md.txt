Актуальная версия на сайте https://www.ghostscript.com/

Если вдруг не находит путь после установки:

Для Windows:

Перейдите в каталог, в котором установлен GhostScript (например, C:\Program Files\gs\gs9.xx\bin).
Скопируйте путь к этому каталогу.
Щелкните правой кнопкой мыши "Этот компьютер" или "Мой компьютер" и выберите "Свойства".
Нажмите на кнопку "Дополнительные параметры системы".
Нажмите кнопку "Переменные среды".
Под разделом "Системные переменные" прокрутите вниз и найдите переменную "Path", затем нажмите "Изменить".
Добавьте новую запись с путем к каталогу bin GhostScript.
Нажмите "OK", чтобы сохранить изменения.
Перезагрузите компьютер

Для macOS и Linux:

Отредактируйте файл конфигурации профиля вашей оболочки (например, ~/.bashrc, ~/.zshrc).
Добавьте следующую строку в файл, заменив /path/to/gs/bin на фактический путь к каталогу bin GhostScript: 

export PATH="/path/to/gs/bin:$PATH"

Сохраните файл и перезапустите терминал или выполните source ~/.bashrc (или source ~/.zshrc), чтобы применить изменения.
